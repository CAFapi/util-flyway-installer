#### New Features
**US1054145**: The installer now accepts an optional new parameter `db.schema` to specify the database schema to be used. If not specified, the default schema for the database will be used.
**US1053061**: The installer now accepts an optional new parameter `db.useCollation` to specify whether to set C collation when creating the database. Valid options are: [true, false]. Default is false.
